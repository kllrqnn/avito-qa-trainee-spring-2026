# Отчет о найденных дефектах (Bug Report)

---

## BUG-1. Нарушение формата JSON-ответа (Критический)
**Краткое описание:** При создании объявления сервер возвращает ID внутри текстовой строки в поле status, а не в отдельном поле id
**Фактический результат:** `{"status": "Сохранили объявление - <uuid>"}`
**Ожидаемый результат:** `{"id": "<uuid>", "status": "success"}`. Идентификатор должен передаваться в отдельном поле для корректного парсинга автоматизированными системами
**Шаги воспроизведения:** 1. Подготовить валидный JSON-объект для создания объявления
                          2. Выполнить POST запрос на `/api/1/item`
**Серьезность:** Критический
**Окружение:** https://qa-internship.avito.com, macOS, Python 3.12 

---

## BUG-2. Отсутствие валидации отрицательной цены (Минорный)
**Описание:** Сервер возвращает HTTP 200 на запрос с отрицательной ценой
**Фактический результат:** Статус-код 200 OK, объявление создано
**Ожидаемый результат:** HTTP 400 Bad Request
**Шаги воспроизведения:** 1. Отправить POST запрос на эндпоинт `/api/1/item`
                          2. В теле запроса передать корректный JSON, но с отрицательной ценой:

```json
{
"name": "Negative Price Item",
"price": -100,
"sellerId": 999999,
"statistics": {
    "contacts": 1,
    "likes": 1,
    "viewCount": 1
    }
}
```
**Серьезность:** Минорный
**Окружение:** https://qa-internship.avito.com, macOS, Python 3.12

## BUG-3. Расхождение в именовании полей (Мажорный)
**Описание:** В объекте статистики поле лайков называется `likes`, тогда как в первичной документации/коллекции оно указано как `like`
**Фактический результат:** Ошибка 400 при использовании `like`
**Ожидаемый результат:** Унификация нейминга
**Шаги воспроизведения:** 1. Отправить POST запрос на эндпоинт `/api/1/item`
                          2. В теле запроса передать корректный JSON, используя поле like внутри объекта статистики:

```json
{
"name": "Test Item",
"price": 100,
"sellerId": 123456,
"statistics": {
    "contacts": 10,
    "like": 5,
    "viewCount": 100
    }
}
```
**Серьезность:** Мажорный
**Окружение:** https://qa-internship.avito.com, macOS, Python 3.12
